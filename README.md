<h2>Como executar?</h2>

1- Insira o arquivo .sh no seu $HOME directory<br>
2- Dê a permissão: <pre>chmod 777 linux-essentials.sh</pre>
3- Em seguida execute: <pre>./linux-essentials.sh</pre>
